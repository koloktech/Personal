# MeemCode — Muhammad Syamim Yakop

Personal portfolio site for a Cyber-Physical Systems Engineer (BEM-registered Mechanical / Research Engineer) working across mechanics, software, and applied AI.

## Stack
Static HTML + CSS + vanilla JS. No build step. Dark/light mode, animated terminal hero, scroll reveals.

## Structure
- `index.html` — markup / content
- `styles.css` — theme tokens, layout, components
- `script.js` — theme toggle, scroll reveal, year stamp

## Run
Open `index.html` in a browser, or serve with any static server:
```bash
npx serve .
```

## Sections
Hero · About · Pillars (Mechanical · Software/AI · Connectivity) · Projects · Skills · Experience · Contact
